# SanDiego_-airbnb_pricing_analysis
## Business Problem
San Diego hosts leave money on the table by not 
understanding optimal pricing strategies. This analysis 
identifies overpriced and underpriced neighbourhoods, 
superhost pricing advantages, and what factors drive 
premium pricing across numerous listings.

## Key Findings
→ Analysed 13,162 San Diego listings to identify
  pricing strategies across 50+ neighbourhoods
→ Discovered superhosts earn 4.88 avg review vs
  4.71 for regular hosts despite charging 4x less
→ Identified Carmel Mountain as top ROI neighbourhood
  for property investors

## SQL Concepts Demonstrated
→ Window Functions (RANK, ROW_NUMBER, LAG, AVG OVER)
→ CTEs for multi step analysis
→ JOINs across listings and reviews tables
→ CASE WHEN for data classification
→ Aggregate functions with GROUP BY and HAVING

## Tools Used
→ PostgreSQL 18
→ pgAdmin
→ Inside Airbnb Dataset

## Dataset
Source: Inside Airbnb (insideairbnb.com)
City: San Diego, California
